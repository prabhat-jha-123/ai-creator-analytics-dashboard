# AI Creator Analytics Dashboard

A premium analytics dashboard built using Python, HTML, CSS, JavaScript, and Chart.js.

## Features

- Creator analytics visualization
- YouTube performance tracking
- Dynamic chart rendering
- JSON-based data integration
- Responsive dark UI
- Analytics-focused dashboard design

## Technologies Used

- Python
- HTML
- CSS
- JavaScript
- Chart.js

## Dashboard Metrics

- 108M+ Total Views
- 260K+ Subscribers
- 4 Channels Managed
- Audience CTR Tracking

## Purpose

This project was created to explore creator analytics systems, dashboard visualization, and frontend-based performance monitoring workflows.
